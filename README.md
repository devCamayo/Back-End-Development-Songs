# Back-End-Development-Songs

## 🎵 Servicio de Canciones - Backend

Este servicio proporciona una API RESTful para la gestión de canciones, desarrollada con Flask. Permite crear, leer, actualizar y eliminar canciones a través de endpoints REST. Es parte del proyecto final del curso IBM Back-End Development.

---

## 🚀 Configuración del Entorno

### Requisitos Previos
- **Python 3.9.18** (versión exacta: 3.9.18)
- **Git** (para control de versiones)
- **pip** (gestor de paquetes de Python)
- **curl** (para pruebas de endpoints)

### Creación a partir de la Plantilla

Este proyecto ha sido creado siguiendo la plantilla estándar del curso IBM Back-End Development. La estructura del proyecto incluye:

- `app.py` - Servicio Flask con endpoints REST para canciones
- `test_app.py` - Pruebas unitarias con pytest
- `requirements.txt` - Dependencias del proyecto
- `Dockerfile` - Configuración para contenedor Docker
- `.s2i/environment` - Configuración para OpenShift S2I
